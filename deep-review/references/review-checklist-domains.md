# Review Checklist: Domain Index

Domain-specific checklists are split into one file per domain. Prefer reading only the relevant files.

- `domains/api-backend.md`: API / backend services
- `domains/ui.md`: user interfaces
- `domains/nextjs-react-typescript.md`: Next.js / React / frontend TypeScript
- `domains/rails-ruby.md`: Rails / Ruby / RSpec
- `domains/automation.md`: automation, review, and CI workflows
- `domains/platform-pr-review.md`: platform PR/MR review posting and moderation workflows
- `domains/external-commands-artifacts.md`: external commands and generated artifacts
- `domains/event-driven-realtime.md`: event-driven, realtime, and streaming workflows
- `domains/integrations.md`: external integrations
- `domains/data-jobs.md`: data stores, background jobs, and batch work
- `domains/media-time-series.md`: media, time-series, and ordered data
- `domains/go-backend.md`: Go backend services
- `domains/tests.md`: test quality and regression coverage
